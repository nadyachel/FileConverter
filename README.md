# Универсальный Конвертер Файлов (File Converter)

Это приложение позволяет осуществлять пакетную конвертацию файлов различных форматов. Разработано в рамках лабораторной работы №3.

## Функциональность
- Пакетная конвертация файлов из выбранной директории.
- Поддержка изображений (PNG, JPG, WEBP).
- Поддержка текстовых данных (CSV в JSON).
- Настройка параметров конвертации (например, качество для JPEG-изображений).

## Инструкция по установке и запуску
1. Клонируйте репозиторий:
   `git clone <url_репозитория>`
2. Создайте и активируйте виртуальное окружение:
   `python -m venv venv`
   В Windows: `venv\Scripts\activate`
   В Linux/Mac: `source venv/bin/activate`
3. Установите зависимости:
   `pip install -r requirements.txt`
4. Запустите приложение:
   `python main.py`

## Использованные технологии
- **Язык:** Python 3.10+
- **Библиотеки:** Pillow (для работы с изображениями), стандартные библиотеки `os`, `json`, `csv`.
- **Архитектура:** Объектно-ориентированное программирование (ООП), паттерны проектирования.

## Архитектура и паттерны
В проекте использованы следующие паттерны:
1. **Strategy (Стратегия)** - интерфейс `IConverterStrategy` позволяет легко добавлять новые форматы без изменения существующего кода.
2. **Factory (Фабрика)** - `ConverterFactory` инкапсулирует логику выбора нужного конвертера на основе расширений файлов.
3. **Facade (Фасад)** - `BatchConverterFacade` скрывает сложную логику обхода директорий и вызова нужных фабрик/стратегий, предоставляя пользователю один простой метод.
